Oparation Starfall
Magneic platform

An couch co-op Game made by NeoN Games Studio With students

the team for this project consisted of 20+ members. My part in the project, along with a classmate, was to make new game mechanism. finally, we made a magnetic platform that pulls the drones toward itself whenever the drone is close by.

In this Project, I was very able to improve my understanding of C# and I learned a lot about Version Control and Code Convension, which makes the code cleaner and easier to read. I myself was struggling a lot in this project with the Code for the force System and the pace at which the Drone is being pulled onto the Platform.

The method OnEnter() is invoked when the script is first enabled. It verifies whether the target Magnetic Platform Object is a viable object, and if so, it triggers a state change in the Target Object to a "MagneticHoldState". It also establishes the magnetic platform object as the MagneticPlatform property of the MagneticHoldState of the target object.

The WaitForTransition() Coroutine waits a specified time then sets a Trigger in the state machine of the object to signal that the target object has been pulled successfully to the Magnetic Platform.

The method PullTarget() then calculates the Direction and magnitude of the force to be applied to the targeted object according to its position relative to the magnetic platform. It then adds this force to the ForceBody component of the targeted object, simulating physics and motion for the object.

The method IsCorrectPosition() verifies whether the target object is located within a certain distance range from the magnetic platform. If so, "true" is returned, which indicates that the target object is in the correct position.
